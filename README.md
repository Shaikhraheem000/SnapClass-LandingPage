# SnapClass Landing Page

This repository contains the official landing page for SnapClass, an AI-powered attendance management system. The page introduces the product, explains the teacher and student workflows, and links users to the deployed Streamlit application.

## Live Links

- Landing page: https://snap-class-landing-page-zeta.vercel.app/
- Main app: https://snappclass.streamlit.app/
- Main app repository: https://github.com/Shaikhraheem000/Snapp-Class

## Features

- Clean product landing page
- Hero section with call-to-action buttons
- Feature cards for face attendance, voice attendance, and QR enrollment
- Teacher workflow section
- Student workflow section
- Tech stack section
- Demo screenshots from the SnapClass app
- Responsive layout for desktop and mobile

## Tech Stack

## Setup Instructions

1. Clone the repository:

```bash
git clone https://github.com/Shaikhraheem000/SnapClass-LandingPage.git
cd SnapClass-LandingPage
```

2. Create and activate a virtual environment:

```bash
python -m venv venv
venv\Scripts\activate
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run locally:

```bash
python app.py
```

5. Open in browser:

```text
http://localhost:5002
```
